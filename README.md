# ✅ Трекер привычек — Habit Tracker

**Версия:** 1.0.2
**Язык:** Python 3.12
**Хранение данных:** CSV (встроенный модуль `csv`)
**Тестирование:** pytest + pytest-cov

---

## Описание

Консольное приложение для ежедневного учёта выполнения личных привычек
(спорт, чтение, медитация и т.д.), расчёта текущей серии (streak) и
контроля прогресса по недельной цели. Реализует полный цикл разработки:
от требований и TDD до приёмочного тестирования и сопровождения.

## Структура данных (CSV)

Каждая строка — это одна отметка о выполнении привычки в конкретный день.

| Поле             | Тип   | Описание                                          |
|------------------|-------|----------------------------------------------------|
| id               | int   | Уникальный идентификатор записи                    |
| date             | str   | Дата выполнения (YYYY-MM-DD)                       |
| habit_name       | str   | Название привычки                                  |
| category         | str   | Категория (Спорт, Здоровье, Развитие, ...)         |
| target_per_week  | int   | Целевая частота выполнения, раз в неделю (1–7)     |
| note             | str   | Произвольная заметка (необязательно)               |

---

## Расчёт серии (streak) и недельного прогресса

- **Streak** — количество последовательных дней (считая назад от указанной
  даты, по умолчанию от сегодня), в которые привычка была отмечена выполненной.
  Серия обрывается на первом пропущенном дне.
- **Недельная сводка** — для недели, начинающейся с указанной даты (7 дней),
  считается количество выполнений и процент от цели `target_per_week`.
  Если выполнений больше цели — процент может превышать 100%.

## FAQ

**Q: Как сбросить все данные?**
Удалите файл `data/habits_log.csv` или выполните:
```python
python -c "import data_service; data_service.clear_all()"
```

**Q: Можно ли указать цель словами вместо числа?**
Да, в меню при вводе цели можно написать «каждый день» или «daily» — это
будет распознано как 7 раз в неделю.

**Q: Что если я отметил привычку дважды за один день?**
Создаются две отдельные записи с одинаковой датой — это не влияет на расчёт
streak (он учитывает уникальные дни), но влияет на статистику количества
выполнений за период.
